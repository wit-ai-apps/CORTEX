## Claudeの問い

Cursorは「BのコアをDが再利用できる」と言った。

では逆に聞く。

```
DのコアをBが再利用できないのか？
```

具体的にはこういうことだ。

```
Step1：D（GitHub Actions）を先に作る
  ↓
  ローカルで同じチェックを回したくなる
  ↓
Step2：B（CLI）は D のラッパーとして後からつく
```

この順序でも技術的に成立するはずだ。

**BがDの土台になれるなら、DもBの土台になれる。**  
これはBとDのどちらを先にする理由にもならない。
